# 기상과 지하철 열차 혼잡도 간 상관관계 분석 및 예측

## 프로젝트 개요
본 프로젝트는 **기상 데이터와 지하철 혼잡도의 상관관계를 분석**하고, 딥러닝 모델을 활용하여 **1시간 뒤의 특정 지하철역 혼잡도를 예측**하는 것을 목표로 합니다.
- **사용 데이터:** 날씨마루 분석 플랫폼 제공 기상 데이터 및 지하철 혼잡도 원본 데이터
- **예측 방식:** 현재 시간을 포함한 총 2시간의 데이터를 이용하여 1시간 뒤의 혼잡도 예측

## 데이터 전처리 및 주요 변수
### 주요 특징(Features)
- `station_number` (역번호)
- `direction` (열차 방향: 0=상선, 1=하선, 2=내선, 3=외선)
- `temp` (기온), `wind_speed` (풍속), `rn_hr` (시간강수량), `humidity` (상대습도), `solar` (일사량)
- `feel_temp` (체감온도), `feel_stage` (체감온도 범주화)
- `Is_holiday` (휴일 여부: 0=평일, 1=주말/공휴일)

### 결측치 처리
- **일사량:** 일몰/일출 기준 결측치 0으로 변환 (여름, 겨울, 봄/가을 구분)
- **보간(Interpolation):**
  - 0단계: 동일 관측소 내 시간순 `bfill`, `ffill` 적용 (풍향, 강수량 등)
  - 1단계: 선형 보간 (기온, 풍속, 습도, 일사량, 체감온도 등)
  - 2단계: 남은 결측치는 그룹 내 중앙값으로 보간
